Interesting things I've stumbled upon recently.

## Web dev & User interface

* [https://htmx.org/](https://htmx.org/) - I've seen a bit too many hours get wasted fixing webpack configurations, so I'm looking forward to using something like HTMX or [Elixir LiveView](https://hexdocs.pm/phoenix_live_view/) to simplify some of my upcoming web UI work.

* [Smartcrop.js](https://github.com/jwagner/smartcrop.js): Node & browser smart cropping library

* [Stitches](https://stitches.dev): CSS-in-JS library with near-zero runtime, SSR, multi-variant support

* [Theatre.js](https://www.theatrejs.com/): a JavaScript animation library with a GUI

* [kbar](https://kbar.vercel.app): An extensible `command+k` interface for websites

* [7guis](https://eugenkiss.github.io/7guis/): A GUI Programming Benchmark

* [React from scratch](https://react-from-scratch-slides.netlify.app/#3): Some interesting bits here even for somewhat experienced react developers

* [Mantine](https://mantine.dev): Complete UI component lib for React

## Other dev stuff

* [apscheduler](https://github.com/agronholm/apscheduler): Python scheduler library which seems simple to use for a change

* [spaceship-prompt](https://github.com/spaceship-prompt/spaceship-prompt): A Zsh prompt 'for astronauts'

* [starship](https://starship.rs/): A rust-based customizable prompt for any shell

* [The Missing Semester of Your CS Education](https://missing.csail.mit.edu): How to master the command-line, use a powerful text editor, use fancy features of version control systems, and much more!
