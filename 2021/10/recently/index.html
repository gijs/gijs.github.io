Interesting things I’ve stumbled upon recently.

Web dev & User interface

  • https://htmx.org/ - I’ve seen a bit too many hours get wasted fixing webpack configurations, so I’m looking forward to using something like HTMX or Elixir LiveView to simplify some of my upcoming web UI work.

  • Smartcrop.js: Node & browser smart cropping library

  • Stitches: CSS-in-JS library with near-zero runtime, SSR, multi-variant support

  • Theatre.js: a JavaScript animation library with a GUI

  • kbar: An extensible command+k interface for websites

  • 7guis: A GUI Programming Benchmark

  • React from scratch: Some interesting bits here even for somewhat experienced react developers

  • Mantine: Complete UI component lib for React

Other dev stuff

  • apscheduler: Python scheduler library which seems simple to use for a change

  • spaceship-prompt: A Zsh prompt ‘for astronauts’

  • starship: A rust-based customizable prompt for any shell

  • The Missing Semester of Your CS Education: How to master the command-line, use a powerful text editor, use fancy features of version control systems, and much more!

Compsci

  • Neural networks from the ground up: Neural networks explained in a way that’s easy to understand

Other things
