Now you know what a [tesseract](http://en.wikipedia.org/wiki/Tesseract) is, here's [Tesseract](http://square.github.com/tesseract/), a Javascript library for filtering large multivariate datasets in the browser. 

The project uses D3.js (ofcourse), and can be downloaded or forked from [Github](https://github.com/square/tesseract/).

**DOH! the project was renamed to [Crossfilter](https://github.com/square/crossfilter).**
