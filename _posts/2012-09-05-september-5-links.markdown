 * [Spark](http://www.meteor.com/blog/2012/08/31/introducing-spark-a-new-live-page-update-engine) can be seen as a declarative version of jQuery. jQuery is imperative. To get work done in jQuery, you tell it to carry out a series of actions: "Find these nodes. Add this class to them. Replace this thing with that." Spark is declarative. You tell it how you want the page to look, and not only does it make the page look that way, it keeps the page updated as data changes. Spark weighs in at 8k gzipped and minified, including all of its dependencies, and it's easily separable from the rest of [Meteor](http://www.meteor.com).
