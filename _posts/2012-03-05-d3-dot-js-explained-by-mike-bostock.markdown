Mike Bostock is the author of the excellent [D3.js library](http://mbostock.github.com/d3/). If you want to learn D3, or interactive visualization in general, better listen to him.

These three presentations are the de-facto standard documentation for D3:

 * [Thinking with Joins](http://bost.ocks.org/mike/join/)

 * [D3: Visualizing Data with Web Standards](http://mbostock.github.com/d3/talk/20111116/#0)

 * [D3 Workshop](http://bost.ocks.org/mike/d3/workshop/)
