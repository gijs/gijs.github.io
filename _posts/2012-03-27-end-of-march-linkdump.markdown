Perpetual Ocean
---------------


![](/images/perpetual_ocean.jpg)

[Perpetual Ocean](http://svs.gsfc.nasa.gov/vis/a000000/a003800/a003827/) shows ocean surface currents around the world during the period from June 2005 through Decmeber 2007. The visualization does not include a narration or annotations; the goal was to use ocean flow data to create a simple, visceral experience.


Elastic Search
--------------
[Elastic Search](http://www.elasticsearch.org/) is an opensource, distributed, RESTful searchengine on top of [Apache Lucene](http://lucene.apache.org/). See [Github](https://github.com/elasticsearch/elasticsearch).


QGIS basics for journalists
---------------------------
From the [Knight Digital Media Center](http://multimedia.journalism.berkeley.edu/) comes [this excellent QGIS walkthrough](http://multimedia.journalism.berkeley.edu/tutorials/qgis-basics-journalists/). It covers projections, vector maps, simplification, analysis, exporting and some more. 

Another interesting tutorial is [this one](http://multimedia.journalism.berkeley.edu/tutorials/google-refine-export-json/) on how to use [Google Refine](http://code.google.com/p/google-refine/) to export arbitrary data to JSON.


Raphael.js and D3.js used to visualize Global Agenda Survey 2011
----------------------------------------------------------------
[Global Agenda Survey 2011](http://reports.weforum.org/global-agenda-survey-2011/) was built with [Raphael.js](http://raphaeljs.com/) and [D3.js](http://mbostock.github.com/d3/). 


Inkscape and D3.js
------------------
[D3.js](http://mbostock.github.com/d3/) and [Inkscape](http://inkscape.org/) are a powerful information visualization couple, and [this tutorial](http://christopheviau.com/d3_tutorial/d3_inkscape/) explains some details of how to integrate them into a workflow.


Big-ish Data with node.js, SQLite, Redis and Polymaps
-----------------------------------------------------
[A presentation](http://macwright.org/presentations/nodedc/#1) by [Tom MacWright](https://twitter.com/#!/tmcw) about using [node.js](http://nodejs.org/), [SQLite](http://www.sqlite.org/), [Redis](http://redis.io/) and [Polymaps](http://polymaps.org/).


SQL.js
------
Not sure if I've covered this before but [SQL.js](https://github.com/kripken/sql.js) is [SQLite](http://www.sqlite.org/) ported to Javascript using [Emscriptem](https://github.com/kripken/emscripten/wiki). Check out [the demo](http://syntensity.com/static/sql.html).


Javascript Shapefile-to-GeoJSON
-------------------------------
[Convert Shapefiles and DBF to GeoJSON using JavaScript](https://github.com/wavded/js-shapefile-to-geojson)

This project allows a user to load Shapefiles and DBFs into the browser with JavaScript. Outputs as GeoJSON for use with other Mapping APIs such as OpenLayers.
