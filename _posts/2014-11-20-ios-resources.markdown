 * [TheClimateCorporation/TCCMapTileAnimation](https://github.com/TheClimateCorporation/TCCMapTileAnimation) is an iOS project to display animated map tiles within MapKit.

 * [mbxmapkit](https://www.mapbox.com/mbxmapkit/) is a small open source library for building map applications on the Apple platform. 

 * [facebook/AsyncDisplayKit](https://github.com/facebook/AsyncDisplayKit) is an iOS framework that keeps even the most complex user interfaces smooth and responsive. It was originally built to make Facebook's Paper possible, and goes hand-in-hand with pop's physics-based animations — but it's just as powerful with UIKit Dynamics and conventional app designs. [More](http://asyncdisplaykit.org).

 * [AFNetworking/AFNetworking](https://github.com/AFNetworking/AFNetworking) ([afnetworking.com](http://afnetworking.com)) is an iOS and OS X networking framework.

 * [jverdi/JVFloatLabeledTextField](https://github.com/jverdi/JVFloatLabeledTextField) is the an implementation of a UX pattern that has come to be known the "Float Label Pattern".

 * [Masonry/Masonry](https://github.com/Masonry/Masonry) allows easier use of AutoLayout NSLayoutConstraints with a simplified, chainable and expressive syntax.

 * [romaHerman/timeLineiOS](https://github.com/romaHerman/timeLineiOS) is a drop-in timeline with progress animatiom.

 * [mineschan/MZTimerLabe](https://github.com/mineschan/MZTimerLabel) is a handy class for using UILabel as a countdown timer or stopwatch just like in Apple Clock App.

 * [zhxnlai/ZLSwipeableView](https://github.com/zhxnlai/ZLSwipeableView) is simple view for building card like interface like Tinder and Potluck.

 * [jonathantribouharet/JTCalendar](https://github.com/jonathantribouharet/JTCalendar) is a customizable calendar view for iOS.

 * [Raizlabs/RZTransitions](https://github.com/Raizlabs/RZTransitions) is a library to help make iOS7 custom View Controller transitions slick and simple.

 * [carsonperrotti/CNPGridMenu](https://github.com/carsonperrotti/CNPGridMenu) is a Mailbox style grid menu with a blurred background.

 * [DSFacialGestureDetector for iOS](https://www.cocoacontrols.com/controls/dsfacialgesturedetector)

 * [filippocamillo/FCVerticalMenu](https://github.com/filippocamillo/FCVerticalMenu) is a vertical menu animated and fully customizable.
