Tabata Pro
----------
[Tabata Pro](http://itunes.apple.com/us/app/tabata-pro-tabata-timer/id346432063?mt=8) is a timer application for use with the [Tabata training protocol](http://journals.lww.com/acsm-msse/Abstract/1996/10000/Effects_of_moderate_intensity_endurance_and.18.aspx) also known as High Intensity Interval Training.
I've only used it for a training or ten, but I can say it works perfectly. It's definitely as good as the famous Gym Boss timers.


Wild Chords
-----------
[Wild Chords](http://itunes.apple.com/us/app/wildchords/id453707538?mt=8) is a free guitar learning app. I actually can't believe it's free, because it's brilliant. In a playful way, it takes beginning guitar players through various levels where you're challenged to play chords in order to entice all kinds of animals that've fled the zoo and lead them back.
Also, its built-in tuner is truly genius. Highly recommended for beginning guitar players like myself. If you're not convinced, have a look at [Ovelin's pitch](http://www.youtube.com/watch?v=lSivD-G0nQI) for the game at Slush2011.


Aelios
------
[Aelios](http://itunes.apple.com/nl/app/aelios-weather/id436105028?mt=8) is a polished weather app. It has quite a unique user interface which isn't always the easiest, but it's pretty and it works. It gets updated now and then, and useful new features like bookmarking locations were just added.


Prompt
------
[Prompt](http://itunes.apple.com/us/app/prompt/id421507115?mt=8) is a Terminal client which can connect via SSH and supports color and fancy key combinations. Can be very handy for remote work, especially if you attach a bluetooth keyboard.


Penultimate
-----------
[Penultimate](http://itunes.apple.com/nl/app/penultimate/id354098826?mt=8) is a drawing/note-taking app. It was the first app I bought. I've used it in the beginning, and found it to be ideal for quick sketches, not so much for text.


Air Video
---------
[Air Video](http://itunes.apple.com/nl/app/air-video-watch-your-videos/id306550020?mt=8) can stream video from your Mac to iOS devices. It needs a 'server' package on the Mac side, which is a free download. It can transcode streams on-the-fly from pretty much every format you can find. One thing it cannot do (yet) is streaming of (physical) DVD's.


GarageBand
----------
[GarageBand](http://itunes.apple.com/pl/app/garageband/id408709785?mt=8). What else do I need to say? I wasn't convinced when I read about it, you just have to experience it. Excellent value for money if you're at least a bit interested in music production.


iMockups
--------
[iMockups](http://itunes.apple.com/nl/app/imockups-for-ipad/id364885913?mt=8) is a tool for quick user interface sketching using pre-fab elements like input fields and windows. It's quite limited and I prefer doing mockups in straight HTML and CSS these days. Still, it's a useful app that I would recommend especially to non-technical people who want to communicate their idea of an interface.


SketchBook
----------
[SketchBook Pro](http://itunes.apple.com/nl/app/sketchbook-pro-for-ipad/id364253478?mt=8) is a full featured drawing app. [Gallery here](http://www.flickr.com/groups/aliassketchbook/). It has layers and loads of brushes, color management et cetera. Recommended even if you're only drawing snow men while riding the train.
 

Pages
-----
[Pages](http://itunes.apple.com/nl/app/pages/id361309726?mt=8) is the only of the three iWork for iOS apps ([Keynote](http://itunes.apple.com/nl/app/keynote/id361285480?mt=8) and [Numbers](http://itunes.apple.com/nl/app/numbers/id361304891?mt=8)) I've bought, and it doesn't disappoint. It does everything I expect of a wordprocessor. Since iCloud support was added, even the syncing problems have disappeared.


iMAME
-----
[iMAME](http://itunes.apple.com/nl/app/imame/id485639079?mt=8) is an Arcade emulator ([MAME](http://mamedev.org/)) formerly available only on jailbroken devices. [More here](http://www.engadget.com/2011/12/21/imame-emulation-app-hits-the-app-store-humanity-cheers-in-uniso/)


Snapseed
--------
[Snapseed](http://itunes.apple.com/nl/app/snapseed/id439438619?mt=8) is a Photo retouching app from Nik Software. They build expensive Photoshop plugins too, and this shows in the quality of Snapseeds filters and effects. The touch interface is a pleasure to use, especially on the iPad. Next level interaction design in my opinion.
