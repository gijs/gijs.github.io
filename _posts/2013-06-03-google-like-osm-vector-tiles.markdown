OK, maybe not as pretty or advanced as the new [Google Maps](https://www.google.com/maps/preview), but still...

[![Image](/images/osm_vector_demo.jpg "OSM Vectortiles") ](http://gijs.github.io/vectorstyling-demo/)

What is this?

 * No PNG tiles but JSON tiles with vector paths for roads and waterways.
 * Data is directly from (an alternative) OpenStreetMap server.
 * Very responsive, small filesizes.
 * Direct control over the colors and widths of the SVG paths using a GUI controller.
 * The base layer is oldschool PNG tiles.
