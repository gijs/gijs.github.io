Backbone Fundamentals
---------------------
[Backbone Fundamentals](https://github.com/addyosmani/backbone-fundamentals) is a Creative Commons-licensed book about Backbone.js for beginners and advanced users.

It's an initiative by [Addy Osmani](https://github.com/addyosmani), who also runs [a blog](http://addyosmani.com/blog/) with several interesting articles for Javascript developers.

Backbone Fundamentals is a great resource if you don't know where to start or if you find the typical todo-list examples too limited.

I found the following parts especially interesting:

 * [Decoupling Backbbone](https://github.com/addyosmani/backbone-fundamentals#decouplingbackbone)
 * [Modular Javascript](https://github.com/addyosmani/backbone-fundamentals#modularjs)
 * jQuery Mobile and Backbone.js (in progress)

Introduction to Backbone.js with jQuery Mobile
----------------------------------------------
[Introduction to Backbone.js with jQuery Mobile](http://blog.chariotsolutions.com/2011/12/introduction-to-backbonejs-with-jquery.html) is a recent article about the combination with jQuery Mobile. Comes with an [example project](https://github.com/stevenpsmith/Exercise/tree/BackboneIntro).
Others in this series: [1](http://blog.chariotsolutions.com/2012/01/sorting-collections-with-backbonejs-and.html), [2](http://blog.chariotsolutions.com/2012/01/from-list-to-details-view-using.html) and [3](http://blog.chariotsolutions.com/2012/02/using-jquerymobile-and-backbonejs-for.html).
