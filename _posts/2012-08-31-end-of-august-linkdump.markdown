Github
------

 * [DOMVas](https://github.com/pbakaus/domvas) implements the missing piece that connects the DOM and Canvas. It gives the ability to take arbitrary DOM content and paint it to a Canvas of your choice.
 
 * [SVGeezy](https://github.com/benhowdle89/svgeezy) is a Javascript plugin to handle your images in browsers that don't support SVGs.
 
 * [OSGjs](https://github.com/cedricpinson/osgjs) is a Javascript implementation of [OpenSceneGraph](http://www.openscenegraph.org/projects/osg).
 
 * [MetaQuery](https://github.com/benschwarz/metaquery) offers declarative responsive web design syntax.
 
 * [Coccyx](https://github.com/onsi/coccyx) plugs up Backbone leaks with constructor names and tear-downable view hierarchies.

 * [Darner](https://github.com/wavii/darner) is a very simple message queue server. Hits the middle between RabbitMQ and Redis. Based on [Kestrel](https://github.com/robey/kestrel)
 
 * [TeleHash](https://github.com/quartzjer/TeleHash) is a [Kademlia](http://pdos.csail.mit.edu/~petar/papers/maymounkov-kademlia-lncs.pdf)-based [DHT](http://en.wikipedia.org/wiki/Distributed_hash_table) and JSON protocol for application meshing and eventing 

* [Piecon](https://github.com/lipka/piecon) is a tiny javascript library for generating progress pie charts in your favicon.

 * [tQuery](http://jeromeetienne.github.com/tquery/) is [three.js](https://github.com/mrdoob/three.js/) + [jQuery](http://jquery.com)
 
 * [binaryjs](https://github.com/binaryjs/binaryjs) implements binary websocket streaming in [node.js](http://nodejs.org)
 
 * [smart-time-ago](https://github.com/pragmaticly/smart-time-ago) is a little jQuery library to update the relative timestamps in your document. [Docs](http://pragmaticly.github.com/smart-time-ago/).
 
 * [Facebook Timeline](https://github.com/rmdort/facebook-timeline-html-css) using HTML and CSS
 
 * [BigVideo.js](https://github.com/dfcb/BigVideo.js) is a jQuery Plugin for Big Background Video (and Images).
